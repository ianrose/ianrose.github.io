Overview

I’m a Front-end Designer at NBC News Digital. I take pride in designing and building clean, accessible, creative interactive media. I get pumped about CSS, HTML, typography, mobile development, responsive web design, and the idea that people use the things that I create. Before NBC News I worked in New Orleans at the design studio Thinka and before that the ad agency Zehnder Communications. I’m also the creator of Typesettings and Gridsettings.

Clients

NBC News, Sony, vh1, Disney, VooDoo Music Experience, Hyatt, Visit Baton Rouge, Louisiana Charter Schools

Colophon

This site is built with Hexo, Sass, CSS Flexbox and hosted on GitHub. The body text is set in Leitura News. Supplmentary text is set in Leitura Display. The headings are set in Neue Haas Grotesk Display while the UI elements are in Neue Haas Grotesk. All the fonts are served by fonts.com and the CDN is provided by CloudFlare. Also, thanks to Brad Oyler for all his help.
